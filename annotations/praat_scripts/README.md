# Praat Scripts

This folder contains scripts and notes related to Praat-based annotation workflows.


## Purpose

Scripts in this folder support tasks such as:

- checking annotation files
- supporting segmentation workflows
- maintaining consistent dataset organisation


## Praat Workflow Used

General workflow:

1. Open full recording in Praat.

2. Load as Long Sound object.

3. Identify relevant speech sections.

4. Create TextGrid.

5. Add annotation tiers:

- Part
- Speaker
- Content
- Comments

6. Align transcription with audio boundaries.

7. Review annotations through repeated listening.


## Annotation Review

Quality checks included:

- checking alignment between audio and TextGrid boundaries
- verifying speaker labels
- reviewing unclear segments
- maintaining consistent annotation structure
