# Wakhi Temperature Fieldwork and Audio Annotation Portfolio

This repository documents a semantic-domain fieldwork project on temperature expressions in Wakhi, completed during a Field Linguistics course at Stockholm University. The project combines speech recordings, segmentation, transcription, glossing, metadata organization, and lexical analysis.

## Overview

The data were collected in recorded consultant sessions with a native Wakhi speaker, using scenario-based elicitation, narrative prompts, and follow-up clarification. The focus of the project was to identify how Wakhi expresses temperature across domains such as weather, liquids, body sensation, behaviour, and colour.

## What is included

- Original session recordings.
- Segmented audio clips.
- Praat TextGrid annotations.
- Orthographic, phonetic, glossed, and translated transcripts.
- Metadata tables for sessions and files.
- A temperature lexicon and domain analysis.
- Notes on annotation conventions and methodological decisions.

## File structure

- `audio/full_recordings/` — uncut session recordings.
- `audio/parts/` — segmented clips for individual prompts or topics.
- `annotations/textgrids/` — Praat annotation files.
- `transcripts/` — aligned transcription files.
- `metadata/` — session and file inventories.
- `lexicon/` — lexical entries and domain mapping.
- `docs/` — portfolio report, methodology, and notes.
- `scripts/` — helper scripts for segmentation and validation.

## Naming conventions

Files are named with session date, session number, recording letter, part number, and a short descriptive label.

Example:

- `260211_S3a_P6_Making_Tea.wav`
- `260211_S3a_P6_Making_Tea.TextGrid`

## Annotation conventions

- `//` = broad transcription
- `[]` = narrow transcription
- `''` = loose translation
- `""` = quotation or non-Wakhi term
- `|` = Praat boundary marker

The annotation follows a cautious approach: ambiguous forms are marked conservatively, and uncertain analyses are documented in notes rather than overclaimed.

## Research focus

The project analyses a structured temperature lexicon in Wakhi and investigates how temperature terms are distributed across semantic domains. It also examines change-of-state expressions, degree marking, and the limits of metaphorical extension.
