# Annotation Examples

## Example 1: Temperature Scale

syr → cold  
ʃlut → lukewarm  
ʃund → warm  
mats → hot  
ɣamats → very hot  
ˈtɑːodi mɑts → too hot to drink  

---

## Example 2: Change of State

syr vitk → became cold  
vitɑ / vitja / wost → come to be  

---

## Example 3: Procedural Speech (Tea Making)

Put pot on fire → heat  
Add tea → boil  
Add milk → continue boiling  
Result → ready tea  

---

## Example 4: Non-metaphorical restriction

No use of:
- “cold person”
- “warm heart”
- “heated argument”

Instead:
- baghet = unfriendly person
- karha = anger
