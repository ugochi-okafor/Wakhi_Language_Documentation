# Methodology

## Overview

This project uses field-based linguistic elicitation to study temperature expressions in Wakhi.

Data collection was carried out through:
- structured elicitation
- narrative prompts
- procedural descriptions
- follow-up clarification sessions

---

## Data Source

All data comes from:
- 6 recorded sessions (Feb–Mar 2026)
- single consultant (Sajid Ali Alvi)
- Stockholm University field methods course

---

## Recording Method

- Zoom Handy Recorder H2
- segmented using Praat
- exported as WAV + TextGrid pairs

---

## Analytical Approach

1. Audio segmentation in Praat  
2. Orthographic transcription  
3. IPA approximation  
4. Morpheme-by-morpheme glossing  
5. Semantic domain classification  

---

## Focus

- temperature lexicon structure
- environmental vs non-environmental usage
- change-of-state expressions
- absence of metaphorical extension

---

## Limitations

- single speaker dataset
- controlled elicitation bias
- incomplete phonetic precision in early stages
